# coding-project-template
e-plantShoppinggi

e-plantShopping is a simple React + Redux e-commerce application that allows users to browse plants, add them to a shopping cart, update quantities, and view the total cost before checkout.

Project Overview

This project simulates an online plant shopping experience. Users can:

View a list of available plants

Add plants to a shopping cart

Increase or decrease item quantities

Remove items from the cart

View item subtotals and total cart amount

Continue shopping from the cart page
